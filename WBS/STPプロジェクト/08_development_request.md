### 1-4a. カスタムレポートビルダー ★★★ **【NEW】**

> Salesforceのレポート/ダッシュボード機能に相当。1-4はダッシュボードの「表示」側だけだったが、ここではユーザーが自由にレポートを作る「エンジン」を定義。

| # | やりたいこと | 詳細 |
|---|------------|------|
| 1-4a-1 | レポートタイプ選択 | 「商談レポート」「リードレポート」「活動レポート」等、データソースを選んでレポートを作り始める仕組み |
| 1-4a-2 | カラム選択UI | レポートに表示するフィールドをドラッグ&ドロップで追加・削除・並び替え |
| 1-4a-3 | フィルタビルダー | 「ステージ = 商談中 AND 金額 > 10万 AND 作成日 = 今月」のような条件をGUIで組み合わせ |
| 1-4a-4 | グルーピング（行・列） | データを最大3階層でグループ化（例: 営業担当→月→ステージ）。小計・総計の自動計算 |
| 1-4a-5 | レポートフォーマット切替 | 表形式（Tabular）、サマリー形式、マトリクス（クロス集計）の3種切替 |
| 1-4a-6 | 計算フィールド（数式列） | レポート上で独自の計算列を追加（例: 成約率 = 成約数/商談数）。DBに保存不要のアドホック計算 |
| 1-4a-7 | バケット機能 | フィールドの値を任意のグループに分類（例: 金額を「小口/中口/大口」に分類）。データ変更不要 |
| 1-4a-8 | 条件付き書式 | KPIが閾値を超えたら赤/黄/緑でハイライト |
| 1-4a-9 | ドリルダウン（数字→中身） | ダッシュボード上の全ての数字・グラフをクリック可能。クリックすると「その数字を構成するレコード一覧」に遷移。例:「商談中 8件」→8件の企業名・担当者・金額・ステージ日数の一覧が展開。「MRR ¥95万」→顧客別の月額内訳一覧 |
| 1-4a-9a | 多段ドリルダウン **【NEW】** | 一覧からさらに個別レコードにドリルダウン可能。例:「商談中 8件」→一覧→「山田建設」クリック→案件詳細カード。3階層まで掘れる（集計値→一覧→個別） |
| 1-4a-9b | ドリルダウン時のコンテキスト保持 **【NEW】** | ドリルダウンした際に「どこから来たか」のパンくずリストを表示。ワンクリックで元のダッシュボードに戻れる |
| 1-4a-9c | インラインプレビュー **【NEW】** | ドリルダウンせずに、数字にマウスホバーで構成の概要をツールチップ表示。例:「MRR ¥95万」にホバー→「月額5社: 山田建設¥15万, ケアネット¥15万, ...」がポップアップ |
| 1-4a-10 | ダイナミックフィルタ | ダッシュボード上部にフィルタ（期間/担当者/顧客等）を配置し、全ウィジェットに一括適用 |
| 1-4a-11 | スケジュール配信 | 作ったレポート/ダッシュボードを毎朝/毎週にSlack/メール自動送信 |
| 1-4a-12 | スナップショット（履歴トレンド） | 毎日のKPI値を自動保存し、過去との推移を追跡（例: パイプライン金額の月次推移） |
| 1-4a-13 | レポートフォルダ/共有設定 | レポートを個人用/チーム用/全社用に分類。閲覧権限をロール単位で設定 |
| 1-4a-14 | お気に入り/ピン留め | よく使うレポートをワンクリックでアクセス |
| 1-4a-15 | クロスオブジェクトレポート | リード×商談×契約×運用実績を跨いだ結合レポート |
| 1-4a-16 | CSV/Excelエクスポート | PDF/スライドに加え、CSV/Excelでもデータ出力可能 |
| 1-4a-17 | 埋め込みチャート | 商談詳細ページ等に関連レポートのチャートを埋め込み表示 |

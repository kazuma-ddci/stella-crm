# StellaOS Vision Document

## Core Statement

StellaOSは、会社が取り得る中で最も売上利益が高くなるルートを、データとAIで導き出し、人間が迷わず実行できるようにするAI経営OSである。

StellaOSの目的は、単なるCRM管理や業務効率化ではない。顧客、案件、商材、スタッフ、お金、施策の解像度を上げ続け、一社一社、一人一人、一案件ごとに最適な営業、提案、対応、配置、請求管理を実行できる状態を作る。

## North Star

StellaOSが最大化するものは、会社の売上利益である。

ただし、売上利益の最大化とは、売上だけを追うことではない。受注率、粗利、LTV、継続率、入金安定性、スタッフ生産性、顧客満足度、施策ROIを総合して、会社に最も利益が残る選択をし続けることである。

## StellaOSがなくすもの

- 提案すべき商品を提案できていない機会損失
- 連絡すべき日に連絡できていない追客漏れ
- 商談前に情報が足りないまま臨む非効率
- 顧客ごとの刺さる訴求が分からないまま営業する状態
- 受注確度や粗利を見ずに優先順位を決める状態
- 契約後の対応漏れによる満足度低下
- 請求漏れ、入金確認漏れ、未入金放置
- スタッフの強みや状態と案件配置が合っていない状態
- 入力作業に時間を取られ、本来の営業や顧客対応に集中できない状態

StellaOSがなくすべきものは、単なる作業の無駄ではない。売上利益を減らしているすべての機会損失である。

## Ideal State

StellaOSの理想状態は、次の5つが実現されている状態である。

1. 会社の現実がリアルタイムに近く見える
   メール、LINE、電話、商談、請求、入金、納品、スタッフ行動が自然に集まり、経営者やマネージャーが今の状態をすぐ把握できる。

2. 一社一社への最適解が出る
   この会社には何を提案すべきか、いつ連絡すべきか、どの価格がよいか、誰が担当すべきかを、データとAIが提案する。

3. 人間の入力作業が限りなく減る
   商談記録、CRM更新、日報、週報、請求書、提案書などはAIが下書きし、人間は確認、修正、承認するだけに近づく。

4. 使えば使うほど賢くなる
   受注、失注、返信、継続、解約、入金、スタッフ成果などを学習し、会社ごとの勝ちパターンが蓄積される。

5. 企業、スタッフ、顧客を同時に勝たせる
   会社は売上利益が上がる。スタッフは自分に合った仕事、成長、成果の出し方が見える。顧客は早く、正確で、自分たちに合った提案を受けられる。

## CRMとの違い

従来のCRMは、情報を記録し、案件を管理し、担当者が判断するためのシステムである。

StellaOSは、顧客接点を自動で記録し、AIが情報を整理し、顧客、案件、スタッフ、お金の状態を理解し、次に何をすべきかを提案し、その結果を学習してさらに提案精度を上げるOSである。

StellaOSは、記録するOSではなく、勝つための次の一手を出すOSである。

## The Promise

StellaOSは、企業を勝たせ、顧客を勝たせ、スタッフを勝たせるOSにします。

会社のすべての顧客接点、業務、お金、スタッフの情報をリアルタイムに近く集約し、AIが一社一社、一人一人に対する最適な次アクションを出し続けることで、売上利益を最大化するAI経営OSを目指す。
